# face-mask-detection-
face-mask-detection using cnn 
## Description
This project uses a Convolutional Neural Network (CNN) to detect whether a person is wearing a face mask or not. The model is deployed using Streamlit for image-based prediction.

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Streamlit

## How to Run
1. Clone the repository
2. Install dependencies


```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
